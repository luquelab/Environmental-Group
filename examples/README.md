# Example Input and Output Files

This repository includes two example input files with their respective output results. You can generate these outputs by running the Colab notebook provided in the GitHub README or by accessing the notebook through the project webpage home page.

## Example 1: Methanogens — Mahdis

The first example uses **mcrA protein sequences from methanogens**. The **mcrA gene** encodes methyl-coenzyme M reductase subunit alpha, a key enzyme involved in methane production by methanogenic microorganisms.

This example demonstrates how the pipeline can compare methanogenesis-related protein sequences using physicochemical properties, pairwise sequence similarity, heatmaps, and dendrograms.

## Example 2: Norovirus — Wenyu 

The second example uses **norovirus protein sequences**. Norovirus is a highly contagious virus that commonly causes acute gastroenteritis, and sequence comparison can help examine similarities and differences among viral protein sequences.

This example demonstrates how the same pipeline can be applied to viral protein sequences to generate comparable input-based analyses and visual outputs.

## Example 3: Influenza A H1N1 & H3N2 — Amin 

This project uses M2 protein sequences from Influenza A viruses, including selected H1N1 and H3N2 strains. The M2 gene encodes matrix protein 2, a small membrane protein that functions as a proton channel and plays an important role in viral entry and uncoating.

This example demonstrates how the pipeline can compare influenza M2 protein sequences using physicochemical properties, pairwise sequence similarity, heatmaps, and dendrograms. It also shows how these sequence-based results can be connected to multiple sequence alignment support and structural visualization, including predicted monomeric and tetrameric forms of the M2 protein.
